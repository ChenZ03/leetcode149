Max Points on a Line

Given an array of points where points[i] = [xi, yi] represents a point on the X-Y plane, return the maximum number of points that lie on the same straight line.

Examples :

Example 1


							Input: points = [[1,1],[2,2],[3,3]]
							Output: 3

Example 2


							Input: points = [[1,1],[3,2],[5,3],[4,1],[2,3],[1,4]]
							Output: 4

Solution 1 : The Naive approach

(Without geometry knowledge)

From Example 1 : üßê


							Input: points = [[1,1],[2,2],[3,3]]
							Output: 3

Conditions ? ü§î

4 conditions :


							Going Up => y2 - y1 = postive
							Going Down => y2 - y1 = negative
							Same X axis => x2 == x1
							Same Y axis => y2 == y1

Code Idea for solution 1


						Set a variable (result) to 0
						Loop through the array (i) :
							set 4 temporary variables for 4 conditions

Code Idea for solution 1


						Set a variable (result) to 0
						Loop through the array (i) :
							set 4 temporary variables for 4 conditions
							Initiate 3 more loops 
							Loop 1 : 
								Check if x2 - x1 == y2 - y1 :
									Check if its negative (down if negative ; up if positive)

Code Idea for solution 1


						Set a variable (result) to 0
						Loop through the array (i) :
							set 4 temporary variables for 4 conditions
							Initiate 3 more loops 
							Loop 1 : 
								Check if x2 - x1 == y2 - y1 :
									Check if its negative (down if negative ; up if positive)
							Loop 2 : 
								Check if x2 == x1 (same X axis)
							Loop 3:
								Check if y2 == y1 (same Y axis)
						return Maximum points

Code Idea for solution 2


						Set a variable (result) to 1

Gradient + Hash Map / dictionary

Hash Map ? ü§î

Hash Map ? ü§î

After visiting each coordinates and get the slope , we store it in a dictionary which saves us from visiting the same slope again.

Code Idea for Solution 3

Code Idea for Solution 3


						Check if the length <= 2 : return length 
						Initialize the dictionary, result
						Loop through the array (i) :
							initialize duplicates (check for same coordinates)
							initialize temporary variable 
							Loop through the array (i + 1):
								check if they are the same coordinates
								set gradient key (for hash map)
								+ 1 for each occurance of gradient key
							return most occurance
						return max points
